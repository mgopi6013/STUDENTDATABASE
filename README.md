🎓 Student Database Management System

A command-line based Student Database Management System developed in C to efficiently manage student records. The application provides functionalities to add, view, search, modify, and delete student information while demonstrating core concepts of C programming, file handling, and data structures.


📖 Overview

Managing student records manually can be time-consuming and security management risk is high. This project provides a simple and efficient solution for storing and managing student information through a menu-driven interface.

The system enables users to perform various operations on student records and stores data persistently using file handling techniques.

✨ Features

 ➕ Add new student records
 📋 Display all student records
 🔍 Search student details
 ✏️ Modify existing records
 ❌ Delete student records
 💾 Store records using files
 🧾 Menu-driven interface
 ⚡ Fast and easy record management

📋 Functional Modules

Add Student
Stores a new student record in the database.

View Students
Displays all available student records.

Search Student
Finds a student using a unique identifier.

Update Student
Allows modification of existing student information.

Delete Student
Removes a student record from the database.

Exit
Safely terminates the application.
